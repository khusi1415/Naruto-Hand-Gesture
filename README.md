# Naruto Hand Gesture Recognition

## Overview
A real-time hand gesture recognition project inspired by Naruto. The system detects hand signs using MediaPipe and OpenCV and plays different sound effects based on the detected gestures.

## Features
- Real-time hand detection
- Multiple hand support
- Naruto-inspired hand signs
- Gesture-based sound effects
- OpenCV + MediaPipe integration

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame (for audio)

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
